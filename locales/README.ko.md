# MyGPU: GPU 관리 유틸리티

MyGPU는 NVIDIA `nvidia-smi`의 컴팩트한 랩어웨어로, 우아한 웹 대시보드를 갖춘 가벼운 GPU 관리 도구입니다.

## 언어 지원

- 영어 (영어)
- 독일어 (독일어)
- 프랑스어 (프랑스어)
- 스페인어 (스페인어)
- 일본어 (일본어)
- 중국어 (중국어)
- 포르투갈어 (포르투갈어)
- 한국어 (한국어)
- 힌디어 (힌디어)

## 특징

- **가벼움**: 최소한의 리소스 사용.
- **유연성**: CLI 도구 또는 완전한 웹 대시보드로 실행 가능.
- **관리자 중심**: VRAM 제한 및 자동 종료 기능과 같은 관리자 기능 포함.
- **개발자 친화적**: GEMM 및 입자 물리학 시뮬레이션과 같은 내장 벤치마크 및 스트레스 테스트 도구 제공.

## 대시보드 및 기능

### 웹 대시보드

MyGPU는 직관적인 웹 대시보드를 제공하여 GPU 및 시스템 메트릭을 실시간으로 모니터링할 수 있습니다.

- **GPU 모니터링**: 자원 사용량, VRAM, 전력, 온도 등 상세한 GPU 메트릭.
- **시스템 메트릭**: CPU, RAM 등 시스템 메트릭.
- **관리자 기능**:
  - **VRAM 제한**: GPU당 VRAM 사용량에 대한 하드 한계 설정.
  - **자동 종료**: VRAM 정책을 위반하는 프로세스를 자동으로 종료 (관리자 전용).
  - **워치리스트**: 특정 PID 또는 프로세스 이름을 모니터링.
- **시뮬레이션 및 벤치마크**:
  - **스트레스 테스트**: GEMM 워크로드를 사용하여 열 스로틀링 및 안정성을 테스트.
  - **입자 물리학 시뮬레이션**: GPU 부하를 시각화하기 위한 상호형 3D 입자 시뮬레이션.

### CLI

MyGPU는 명령줄 인터페이스(CLI)를 통해 간편한 모니터링 및 관리 기능을 제공합니다.

## 요구 사항

- **운영 체제**: Windows 10/11
- **Python**: 3.10 이상
- **하드웨어**: NVIDIA GPU 및 설치 드라이버
- **CUDA**: CUDA 12.x (벤치마크 및 시뮬레이션 기능 사용 시 필수)

## 설치

MyGPU는 다양한 요구 사항에 맞게 모듈식 설치를 지원합니다.

### 1. 최소 (CLI 전용)

헤드리스 서버 또는 백그라운드 모니터링에 적합합니다.

- 명령줄 인터페이스만 제공.
- 기본 시스템 및 GPU 메트릭.

### 2. 표준 (CLI + 웹 UI)

대부분의 사용자에게 적합합니다.

- 웹 대시보드 포함.
- REST API 엔드포인트.
- 실시간 차트.
- 하지만 시뮬레이션 또는 벤치마크 없음.

### 3. 전체 (표준 + 시각화)

개발 및 스트레스 테스트에 적합합니다.

- 시뮬레이션 포함.
- PyTorch/CuPy 의존성으로 벤치마크.

### 시작하기

1. **다운로드** 최신 릴리스 또는 저장소를 클론합니다.
2. **설정 실행**:

   ```powershell
   .\setup.ps1
   ```

3. **시작**:

   ```powershell
   # 웹 대시보드 시작 (표준/전체)
   python health_monitor.py web

   # CLI 시작
   python health_monitor.py cli
   ```
