# MyGPU: 轻量级 GPU 管理工具：一个简洁的 `nvidia-smi` 包装器，配有优雅的网络仪表板

## 简介

MyGPU 是一个轻量级的 GPU 管理工具，它是一个紧凑的 `nvidia-smi` 包装器，配有一个精致的网络仪表板。它旨在为用户提供一个简单、高效的方式来监控和管理他们的 GPU 资源。

## 功能

- **实时监控**：提供详细的 GPU 指标（利用率、VRAM、功耗、温度）以及系统指标（CPU、内存等）。
- **管理与强制执行**：
  - **VRAM 限制**：为每个 GPU 设置硬性 VRAM 使用限制。
  - **自动终止**（仅管理员可操作）：当进程超出 VRAM 政策时自动终止。
  - **监控列表**：监控特定 PID 或进程名称。
- **基准测试与模拟**：
  - **压力测试**：配置可配置的 GEMM 工作负载来测试散热和稳定性。
  - **可视化模拟**：交互式 3D 粒子物理模拟，以可视化 GPU 加载。

## 要求

- **操作系统**：Windows 10/11
- **Python**：3.10+
- **硬件**：NVIDIA GPU 和已安装的驱动程序。
- **CUDA**：严格要求 CUDA 12.x（用于基准测试/模拟功能）。
  - *注意：如果未检测到 CUDA 12.x，则基准测试和模拟功能将禁用。*

## 安装

工具支持模块化安装，以适应您的需求：

### 1. 最小化（仅 CLI）

适用于无头服务器或后台监控。

- 命令行界面。
- 基本系统/GPU 指标。

### 2. 标准（CLI + 网络 UI）

适用于大多数用户。

- 包含网络仪表板。
- REST API 端点。
- 实时图表。
- 但不包含模拟或基准测试。

### 3. 完整（标准 + 可视化）

适用于开发和压力测试。

- 包含模拟。
- PyTorch/CuPy 依赖用于基准测试。

### 快速入门

1. **下载** 最新版本或克隆仓库。
2. **运行安装脚本**：

   ```powershell
   .\setup.ps1
   ```

3. **启动**：

```powershell
# 启动网络仪表板（标准/完整）
python health_monitor.py web

# 启动 CLI
python health_monitor.py cli
```
