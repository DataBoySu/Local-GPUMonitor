### इसका उपयोग क्यों करें?

- **हल्का**: न्यूनतम संसाधन पदचिह्न।
- **लचीला**: CLI उपकरण के रूप में या पूर्ण वेब डैशबोर्ड के रूप में चलाएँ।
- **प्रशासक-केंद्रित**: **VRAM प्रवर्तन** (सीमाओं को पार करने वाली प्रक्रियाओं को स्वचालित रूप से समाप्त करें) और **निगरानी सूचियाँ** जैसी सुविधाएँ शामिल हैं।
- **विकासकर्ता के अनुकूल**: जेएमएम (जेनेरिक मैट्रिक्स मल्टीप्लिकेशन) और कण भौतिकी जैसे तनाव परीक्षण और स्थिरता के लिए निर्मित-इन उपकरण।

## विशेषताएँ

- **वास्तविक समय निगरानी**:
  - विस्तृत GPU मेट्रिक्स (उपयोग, VRAM, शक्ति, तापमान)।
  - सिस्टम मेट्रिक्स (CPU, RAM, आदि)।

- **प्रशासक और प्रवर्तन**:
  - **VRAM सीमाएँ**: प्रत्येक GPU के लिए VRAM उपयोग पर कठोर सीमाएँ सेट करें।
  - **स्वचालित समाप्ति**: VRAM नीतियों का उल्लंघन करने वाली प्रक्रियाओं को स्वचालित रूप से समाप्त करें (केवल प्रशासक के लिए)।
  - **निगरानी सूचियाँ**: विशिष्ट PID या प्रक्रिया नामों की निगरानी करें।

- **तनाव परीक्षण और सिमुलेशन**:
  - **तनाव परीक्षण**: जेएमएम लोड के साथ कॉन्फ़िगरेबल GEMM कार्यों का उपयोग करके तापमान थ्रॉटलिंग और स्थिरता का परीक्षण करें।
  - **कण भौतिकी सिमुलेशन**: GPU लोड को दृश्य बनाने के लिए इंटरैक्टिव 3D कण भौतिकी सिमुलेशन का उपयोग करें।

## रोडमैप और भविष्य का कार्य

योगदान स्वागत है! मुख्य भविष्य के बिंदुओं को कवर करना है:

- **बहु-GPU समर्थन**: बहु-कार्ड सेटअप और NVLink टोपोलॉजी के लिए बढ़िया हैंडलिंग।
- **कंटेनरीकरण**: आधिकारिक Docker समर्थन के लिए आसान तैनाती कंटेनराइज्ड वातावरण में।
- **दूरस्थ पहुँच**: SSH टनलिंग एकीकरण और सुरक्षित दूरस्थ प्रबंधन।
- **हार्डवेयर स्वतंत्र**:
  - AMD ROCm समर्थन।
  - Intel Arc समर्थन।
- **बहु-भाषा दस्तावेज़ीकरण**: GitHub पर सबसे लोकप्रिय भाषाओं का समर्थन करना।

## आवश्यकताएँ

- **OS**: Windows 10/11
- **पायथन**: 3.10+
- **हार्डवेयर**: NVIDIA GPU के साथ स्थापित ड्राइवर।
- **CUDA**: टूलकिट 12.x (तनाव परीक्षण/सिमुलेशन सुविधाओं के लिए सख्त रूप से आवश्यक)।

## स्थापना

उपकरण के लिए कई स्थापना विकल्प हैं:

1. **न्यूनतम (केवल CLI)**: सर्वर या पृष्ठभूमि निगरानी के लिए सर्वोत्तम।
2. **मानक (CLI + वेब UI)**: अधिकांश उपयोगकर्ताओं के लिए सर्वोत्तम।
3. **पूर्ण (मानक + सिमुलेशन)**: विकास और तनाव परीक्षण के लिए सर्वोत्तम।

### त्वरित शुरुआत

1. **डाउनलोड** करें या रिपॉजिटरी क्लोन करें।
2. **सेटअप चलाएँ**:

   ```powershell
   .\setup.ps1
   ```

3. **लॉन्च करें**:

   ```powershell
   # वेब डैशबोर्ड शुरू करें (मानक/पूर्ण)
   python health_monitor.py web

   # CLI शुरू करें
   python health_monitor.py cli
   ```
